# Harbor Friend Sample Test

Use this when giving Harbor to a friend for a first click-through test.

## Test Link

Start here:

```text
/sample
```

Then move through:

```text
/ -> /sample -> /login -> /dashboard -> /planner -> /recipes -> /grocery -> /calendar -> /settings -> /billing -> /overlays
```

## What This Sample Is

Harbor is the public blank-canvas family planner product.

This sample uses starter data only. It does not include Grace-HQ, private Grace data, internal dashboards, or company command-center tools.

## What Your Friend Should Test

Ask them to answer these without you explaining the app first:

1. What do you think Harbor does?
2. What did you click first?
3. What page made the most sense?
4. What page felt confusing?
5. Did the app feel like something a normal family could use?
6. What would you expect it to do next?
7. Would you pay for this if it had real login, saved data, and grocery/meal planning working?

## Red Flags To Watch For

- They do not understand what Harbor is within 10 seconds.
- They think this is Grace-HQ or an internal app.
- They cannot find the main demo flow.
- They expect real saved data before storage is connected.
- The navigation feels too technical or too busy.
- The recipe/planner/grocery loop does not feel connected.
