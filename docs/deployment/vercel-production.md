# Harbor Vercel Production

Use the deployable Next.js app from the repository root.

```text
Repository: wspinyanjr87-lab/Harbor-Public
Production branch: main
Root directory: leave blank
Framework preset: Next.js
Install command: npm install
Build command: npm run build
Output directory: leave blank
```

The repository root must contain `package.json`, `package-lock.json`, `app/`, and `next.config.mjs`.

If Vercel reports `/vercel/path0/package.json` missing, confirm that the deployment source is the latest `main` commit and that Root Directory is blank. Do not use a docs-only branch or a nested root path.
